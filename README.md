## 🎮 ▸ WHAT IS THIS GAME?

**Tiki Topple** is a **turn-based strategy web game** where players fight to control a mystical tower of totems.

💀 You don’t just play...
You **manipulate**, **destroy**, and **outsmart** your opponents.

Every move changes the fate of the tower.

---

## ⚡ ▸ GAME MODES

### 🟢 CLASSIC MODE — *Ancient Strategy*

```
⬆️ Rise   💥 Topple   🔥 Banish
```

* Tactical gameplay
* Mind games + planning
* Win by **positioning your totems at the top**

---

### 🔴 CHAOS MODE — *TOTAL ANNIHILATION*

```
☠️ Execute   ⚡ Dominate   🔄 Reincarnate
```

* Brutal mechanics
* No mercy gameplay
* First to **15 points wins**

🔥 Faster. Deadlier. Unpredictable.
